Supabase — Banco de Dados Permanente

Sincronize com um banco de dados PostgreSQL do Supabase para não perder dados ao limpar o navegador e acessar de qualquer dispositivo de forma transparente.

Como configurar o Supabase
  1. Crie uma conta gratuita em supabase.com
  2. Crie um novo projeto (defina o nome e senha do banco)
  3. Vá no menu lateral em SQL Editor e clique em New Query
  4. Cole o script SQL abaixo e clique em Run no canto superior direito
  5. Vá em Project Settings (engrenagem) › API (no menu lateral inferior)
  6. Copie a Project URL e a chave `anon` `public` e cole nos campos acima
✓ Script SQL copiado! Agora cole no SQL Editor do Supabase.
Publicar no GitHub Pages

Para usar com câmera real, hospede no GitHub Pages (HTTPS gratuito):
1. Suba este arquivo para um repositório GitHub
2. Vá em Settings › Pages e ative
3. A câmera funcionará em https://usuario.github.io/repo/
